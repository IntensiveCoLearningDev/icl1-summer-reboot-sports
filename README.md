# 夏日重启计划｜ 21天运动实验

## 介绍

夏天总是一个让人想重新开始的季节。

很多人想运动、想拥有更好的状态，却总是在坚持几天后回到原来的生活节奏。

问题通常不是不知道怎么运动，而是不知道如何找到适合自己的运动方式，并把运动真正融入生活。

因此，我们发起这场为期 21 天的运动实验。

我们不会要求大家跑多少公里、练多少重量，而是希望通过一次共学实验，帮助每个人找到属于自己的运动方式，建立现实可执行的承诺，并逐步形成能够长期运行的运动系统。

在这里：

- 可以跑步
- 可以健身
- 可以骑行
- 可以打球
- 可以攀岩
- 可以散步

重要的不是运动项目，而是重新建立与身体的连接。21 天后，你不一定拥有完美身材，但希望你能够找到一种真正适合自己的运动方式。让运动不再是一项任务，而成为生活的一部分。
## 关键词

运动, 习惯养成, 长期主义, 重启计划

## 学习资料/课程安排
# 第1周｜开始比完美更重要

目标：

让自己先动起来。

讨论：

- 为什么想运动
- 什么阻碍了开始
- 如何降低行动门槛

实践：

建立自己的运动承诺。

---

# 第2周｜不中断比高强度更重要

目标：

学会面对中断，而不是因为一次中断就放弃。

讨论：

- 状态不好怎么办
- 时间不够怎么办
- 如何进行目标降级

实践：

学习使用「降级完成」。

---

# 第3周｜把运动放回生活

目标：

探索长期可持续的运动方式。

讨论：

- 什么运动适合自己
- 如何长期坚持
- 共学结束后如何继续

实践：

设计属于自己的运动系统。

---

# 共学特色活动

## 全国运动接力计划

共同完成全国运动里程挑战。

## 全国运动地图

点亮参与者所在城市。

## 人生第一项运动挑战

尝试一项从未体验过的运动。

## 运动搭子计划

寻找同城运动伙伴。

## 降级完成挑战

练习在困难时依然保持行动。


## 共学激励

运动不是为了成为更厉害的人。而是为了拥有更自由、更有能量的生活。

21 天不需要做到完美。

只需要保持连接。

开始比完美更重要。

不中断比高强度更重要。

希望这个夏天，我们一起找到属于自己的无痛运动系统。
